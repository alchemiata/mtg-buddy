# MTG Buddy

A mobile-first Progressive Web App for Magic: The Gathering table play.

## Features

- Large one-handed life counter with themed land backgrounds
- Permanent tracker with base attack/defense and live modifiers
- Searchable beginner-friendly glossary
- LocalStorage persistence
- Offline-ready PWA manifest and service worker
- Custom fantasy land backgrounds

## Run Locally

Double-click `start-mtg-companion.cmd`, or run:

```powershell
pnpm install
pnpm dev -- --port 5173
```

Then open `http://127.0.0.1:5173`.

## Build

```powershell
pnpm build
```
